# Fake_News_Detection_System_-TRUTHCHECK-_project
TruthCheck is more than just a fake news detector — it is a step toward building a safer and more informed digital ecosystem. By combining AI innovation with an intuitive user experience, the platform empowers users to verify information quickly and confidently in today’s fast-moving online world.


Brief Description 
TruthCheck is an AI-powered web application developed to detect and analyze fake news using Machine Learning and Natural Language Processing (NLP). The project provides users with a modern and interactive platform where they can submit news content and receive real-time authenticity predictions.
The system is built using Python-based backend architecture with organized modules for routes, models, AI detection, database management, and utilities. It also includes a responsive frontend with multiple pages such as Home, About, Dashboard, Contact, Admin Panel, Authentication, and News Detection interfaces.

Objective
The main objective of TruthCheck is to reduce misinformation spread by helping users verify whether online news content is genuine or fake through intelligent AI-based analysis.
